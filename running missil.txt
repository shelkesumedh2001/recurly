  ğŸ”¥ Hot Reload During Development

  What is Hot Reload?

  Hot reload lets you see code changes instantly without restarting the app!

  How to Use:

  1. Make a UI change (change text, colors, add widgets)
  2. Press r in terminal OR Save file (if using VS Code with auto-reload)
  3. See changes instantly (in ~1 second)

  Example Workflow:

  // lib/screens/home_screen.dart
  Text('Monthly Total')  // Original

  // Change to:
  Text('Total This Month')

  // Press 'r' â†’ See change immediately!

  When Hot Reload Works:

  - âœ… UI changes (widgets, layouts, colors)
  - âœ… Text changes
  - âœ… Adding new widgets
  - âœ… Changing widget properties
  - âœ… Updating provider logic (sometimes)

  When You Need Hot Restart (Press R):

  - ğŸ”„ Changing main() function
  - ğŸ”„ Adding new files
  - ğŸ”„ Changing enums
  - ğŸ”„ Changing global variables
  - ğŸ”„ Adding new dependencies

  When You Need Full Restart (Stop & Run Again):

  - ğŸ›‘ Changing models with @HiveType
  - ğŸ›‘ Running build_runner (generating code)
  - ğŸ›‘ Updating pubspec.yaml
  - ğŸ›‘ Changing native code (Android/iOS)
